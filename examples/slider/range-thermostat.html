HTML Range Examples with aria-orientation and aria-valuetext

The following example of a hypothetical online thermostat demonstrates using aria-valuetext and aria-orientation on input[type="range"] controls. The desired temperature is set with the first range control, which is vertically oriented. Two horizontal range controls set fan speed and mode. The range controls illustrate how to use aria-valuetext to provide assistive technologies with meaningful names for numeric values.

Thermostat

25.0°C

Accessibility Features

  • A border around the thumb is used for indicating the range control with keyboard focus.
  • The button elements receive a border style when hovered for users to identify the elements as interactive. Border is not rendered by default since the content of the buttons is primarily as visual labels for values of the range positions. The buttons are only a secondary way to set the value of the range.

Role, Property, State, and Tabindex Attributes

Group

Role Attribute Element Usage
group div Identifies the group of controls representing the thermostat.
aria-labelledby=IDREF div Defines the accessible name for the group containing the controls representing the thermostat.

Range

Role Attribute Element Usage
aria-valuetext input[type=range]
  • A string value that provides a user-friendly name for the current value of the range.
  • For the temperature range, is the value of the range appended with the text °C.
  • For fan and heat/cool, is the text representation of the current state, e.g., Off or Heat.
aria-orientation=vertical input[type=range]
  • Indicates the orientation of the input element.
  • Set to vertical for the temperature range control.

Buttons

Role Attribute Element Usage
tabindex=-1 button
  • The button supports touch access when an external keyboard is not available to operate the range directly.
  • Removes the button element from the tab order of the page.
aria-label=label button Defines the accessible name for each button describing how the associated range will change (e.g. "Heating off", "Fan off" and "Fan auto").
aria-controls=IDREF button
  • Identifies the content on the page that the button controls.
  • Refers to the SVG g element that contains identifies the range control.
